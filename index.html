🤖 啟用 AI 車牌辨識（Cloudflare Worker）

免費方案每天 10 萬次請求，個人使用完全夠用

1

前往 dash.cloudflare.com 免費註冊 → 左側 Workers & Pages → Create → Create Worker

2

把下載的 worker.js 內容貼入編輯器 → Deploy，取得 Worker 網址
xxx.workers.dev

3

Worker 頁面 → Settings → Variables → Add variable
名稱：ANTHROPIC_API_KEY，值：你的 Claude API Key，勾選 Encrypt

4

打開 traffic-report-app.html，找到 Script 第一行：
const WORKER_URL = '...'
換成你的 Worker 網址，重新部署到 Netlify

5

完成！API Key 永遠留在 Cloudflare，手機端看不到

💡 Claude API Key 哪裡拿？
